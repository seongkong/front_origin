# 건설 도면 탐색 프로토타입

## 실행 방법

프로젝트 루트(`front_origin`)에서 아래 명령을 실행합니다.  
`postinstall` 스크립트가 `app` 폴더 의존성을 자동으로 설치합니다.

```
npm install
npm run dev
```

## 기술 스택

- React 19
- TypeScript
- Vite 7
- Tailwind CSS 4

## 구현 기능

- [x] 도면 트리(계층) 탐색 및 도면 선택
- [x] 공종·리비전 목록에서 선택 및 단일 도면 표시
- [x] 공종 겹쳐보기(레이어 온/오프 + 투명도)
- [x] imageTransform 기반 오버레이 정렬(등록점 해석, relativeTo 처리)
- [x] 리비전별 좌표 대응(주민공동시설 건축 등)
- [x] regions가 객체인 경우(101동 구조 A/B) 리비전 평탄화
- [x] 상위 도면에서 하위 도면 영역(polygon) 표시 및 클릭 시 해당 하위 도면으로 이동
- [x] 하위 도면 진입 시 건축 공종 자동 선택
- [x] 「← 상위 도면으로」 버튼으로 상위 도면 복귀
- [x] 리비전 이전/다음 버튼(날짜순 정렬)
- [x] 현재 도면·공종·리비전 컨텍스트 표시

## 미완성 기능

- [ ] 도면 내 텍스트 검색(메타데이터/이미지)
- [ ] 도면·공종 polygon 하이라이트(관심 영역 표시·클릭 확대)
- [ ] 리비전 시각적 타임라인 UI
